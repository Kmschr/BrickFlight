DOWNLOAD: BrickFlight.zip

INSTRUCTIONS:

Extract folder and run the BrickFlight.exe

There's no ingame interface for choosing the Brickadia savefile to load. It currently expects a save named "Brickadia City.brs" to be in your Builds folder

CONTROLS:

  • L - Start Loading Save
  • WASD/Joystick - Plane controls
  • Left Shit - Increase throttle
  • Left Ctrl - Decrease throttle
  • P - Pause plane movement/gravity (helps for getting unstuck)
  • U - Take screenshot (in Pictures folder)
  • F3 - Toggle framerate indicator
  • Numpad 7/9 - Reorient lighting
